# Flipkart Sentiment Analysis (NLP + Streamlit)

A Machine Learning web application that predicts whether a Flipkart product review is **Positive or Negative** using Natural Language Processing.

##  Features
- Predict sentiment of customer reviews

- Trained ML classification model
